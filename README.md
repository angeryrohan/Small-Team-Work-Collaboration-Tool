 <h2>Prerequisites</h2>
 1. firebase installed on PC
 
 <h2>Installation Steps</h2>
 1. Clone the repository 
 <br>

``` 
git clone https://github.com/rohancl/Deltask
```

 2. Go to (Deltamite) folder, start a local server by typing in:
 
 ```
 firebase serve
 ```
